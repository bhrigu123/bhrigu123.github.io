Projects

JEEQuery

http://jeequery.com

A Q&A platform for students preparing for IIT-JEE and other engineering entrance exams.

Built with Django framework and bootstrap, and deployed on AWS EC2 server


Students can post their questions and ask doubts, can use various pre-defined tags, answer questions, upvote, downvote, or even favourite a question

Has 300+ registered users (as of Dec 2016)

DTU Mart

http://dtumart.com

An online portal for my college (DTU) for students to buy, sell, exchange books and materials with each other.


Received great response from the students, with active participation. Also built with Django framework.

Classifier

https://github.com/bhrigu123/classifier

A python module that organizes files in your directories, by classifying them into folders of music, pdfs, images, etc.

Received over 600+ stars on the GitHub repo

Other features include: classifying by date, classifying specific files into specific folders.
